# 🚀 dbt Snowflake Analytics Project

## 📌 Overview
This project demonstrates an end-to-end data transformation pipeline using dbt and Snowflake.

## 🛠️ Tech Stack
- dbt
- Snowflake
- SQL

## 📊 Project Workflow
1. Raw data loaded using dbt seeds
2. Staging models created for cleaning data
3. Final fact table built for analytics
4. Data quality tests applied

## 📈 Key Insights
- Customer-wise total sales
- Number of orders per customer

## ▶️ How to Run
```bash
dbt seed
dbt run
dbt test
